# База знаний компании

Папка используется как источник фактов для LLM в диалогах с клиентом.

Правила:
- добавляйте материалы в `.md` или `.txt`;
- один файл — одна тема (продукт, кейсы, цены, ограничения);
- пишите факты коротко и конкретно.

Система автоматически передает содержимое файлов в AI-контекст при обработке сообщений.
